##Unity on Linux Unity has released some Linux versions which are not officially supported. The latest build can be downloaded form [this forum](http://forum.unity3d.com/threads/unity-on-linux-release-notes-and-known-issues.350256/). Installing it (5.4.0b10+20160317_amd64) on Ubuntu 15.10 went without any issues. It seems Unity is getting serious about the Linux version has they have released several versions recently. And that's good news! ![unity screenshot](http://fireant.github.io/img/unity.png) Now that Unity is ready can continue with the Coursera course on Unity!
